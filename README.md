## 📌 Project Overview

EduPulse Analytics Dashboard is an interactive Excel-based Business Intelligence solution designed to analyze and monitor student performance across multiple dimensions.

The dashboard provides actionable insights into:

- Academic Performance
- Attendance Trends
- Subject Analysis
- Student Distribution
- Stream-wise Performance
- Gender Analysis    
- City-wise Enrollment
- Admission Trends

This project demonstrates advanced Excel Dashboarding skills, data visualization techniques, KPI design, and business-oriented reporting.

---

## 🚀 Business Problem

Educational institutions often struggle to monitor student performance efficiently due to scattered and unorganized data.

This dashboard solves the problem by converting raw student records into meaningful insights that help educators:

✔ Identify weak subjects

✔ Track attendance impact on marks

✔ Monitor pass percentage

✔ Analyze student distribution

✔ Evaluate admission trends

✔ Compare city-wise performance
